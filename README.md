# Book Tracker

**Автор:** Милена Чобан

## Описание программы
GUI-приложение на Python (Tkinter) для учета прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Данные сохраняются в файл `books.json`. Реализована фильтрация списка по жанру и минимальному количеству страниц.

## Инструкция по запуску
1. Убедитесь, что установлен Python 3.6 или выше.
2. Склонируйте репозиторий или скачайте файл `main.py`.
3. Запустите программу командой:
   ```bash
   python main.py
